Predicción de multimorbilidad canina

Dog Aging Project — BIY7121 EV3 CortesDiego
El cuidado preventivo y la actividad del tutor se asocian con la carga de enfermedades del perro.

Guía y diccionario de variables

El modelo predice si el perro presenta multimorbilidad (2 o más condiciones de salud). Ingresa hábitos del tutor y datos del perro; el sistema calcula internamente el preventive_care_score.

Ver diccionario de campos
Campo Qué significa Rango / valores
pa_activity_level Nivel general de actividad física que realiza el tutor con el perro (paseos, juego, ejercicio). 1 (bajo) → 4 (alto)
pa_avg_activity_intensity Intensidad promedio de esas actividades (esfuerzo físico del perro). 0 a 4
total_daily_active_minutes Minutos activos del perro por día (movimiento sostenido, no solo estar despierto). 0 – 600 min
dental_brushing_freq Frecuencia de cepillado dental del perro (hábito preventivo del tutor). 0 (nunca) – 6 (muy frecuente)
daily_supplements Uso de suplementos o frecuencia de examen dental profesional (proxy de cuidado preventivo). 0 – 5
diet_primary Indica si la dieta principal incluye componente orgánico o de mayor calidad percibida. 0 = No, 1 = Sí
weight_kg Peso corporal actual del perro en kilogramos. 1 – 80 kg
age_derived Edad del perro en años. Variable con mayor peso en el modelo entrenado. 0 – 25 años

Resultado: la API devuelve probability (riesgo 0–100%) y multimorbidity_flag (0 = bajo riesgo clasificado, 1 = multimorbilidad probable). Umbral de decisión: 50%. En CSV usa columnas API o nombres DAP (dd_edad_anios, mp_dental_brushing_frequency, etc.).
